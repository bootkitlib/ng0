Layout1 is a layout component with sidenav support. This application is using Layout1 with one primary sidenav and secondary sidenavs. In bellow examples, you can see how to use Layout1 component and how to open/close secondary sidenavs using Layout1Manager service.

Primary Sidenavs

The main sidenav is defined in app.component.html using ng0Layout1Sidenav directive. You can control its properties using Layout1Manager service.

Secondary sidenavs

The main sidenav is defined in app.component.html using ng0Layout1Sidenav directive. You can control its properties using Layout1Manager service.

Header

You can define a header for layout1 using ng0-layout1-header selector. You can also make the header sticky using stickyHeader method.

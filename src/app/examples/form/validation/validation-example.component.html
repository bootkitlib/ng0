Basic Validation

To display validation errors, the form controls must have a name and be bound to a model using ngModel.
Put inputs inside a ng0-form-field to display error messages and apply validation styles.

File Validation

The following validators are available for file inputs:

  • ng0FileMinSize: Minimum file size in bytes.
  • ng0FileMaxSize: Maximum file size in bytes.
  • ng0FileExtension: Allowed file extensions (e.g. .jpg, .png).

Note: These validators require the ng0File directive to be placed on the input element.

Single File

Select one file to see the validation in action. The file size validators will display an error message if the selected file does not meet the specified size requirements. The file extension validator will display an error message if the selected file does not have an allowed extension.

Multiple Files

Select one or more files to see the validation in action. If multiple files are selected, the validators will check each file against the specified criteria and display an error message if any file does not meet the requirements.

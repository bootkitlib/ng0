This example demonstrates how to use formatters in your application.
Formatters are used to format data for display purposes, such as dates, numbers, and currencies.
Field formatter

A field formatter formats an object by extracting a specific field from it.
This example shows how to use a formatter to display the 'name' field of an object.

Value = {{fieldExample.value | json}}
Formatter = 'name'
Formatted Value = {{fieldExample.value | ng0Format:'name'}}
Array index formatter

An array index formatter formats an array by extracting the element at a specific index.
This example shows how to use a formatter to display the element at index 1 of an array.

Value = {{arrayIndexExample.value | json}}
Formatter: 1
Formatted Value = {{arrayIndexExample.value | ng0Format: 1}}
Enum Formatter

An enum formatter formats an enum value into a human-readable string.
This example shows how to use a formatter to display the string representation of an enum value.

Value = {{enumExample.value | json}}
Formatter = Sexuality
Formatted Value = {{enumExample.value | ng0Format: '@Sexuality'}}

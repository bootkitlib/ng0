Filter

To filter the options in the select component, set filterable to true and provide a filter predicate.
You can pass the name of the property to filter by using the filterBy input or a custom filter function. by default, the select component will filter item values using a default filter function.

[source]="stringArray"; filterable="true" Value: {{ filterExample.value1 | json }}
[source]="personList1"; filterable="true"; formatBy="name"; filterBy="name" Value: {{ filterExample.value2 | json }}
[source]="personList1"; filterable="true"; formatBy="name"; [filterBy]="filterExample.filterFunc" Value: {{ filterExample.value3 | json }}

Format items

Format the display of items in the dropdown and in the input field. To format a value, you need a formatter function.
There is a set of built-in formatters available, or you can create your own custom formatters.

Some built-in formatters:
  • Field formatter
  • Number formatter
  • Currency formatter
  • Locale formatter
To see the list of available built-in formatters, check the formatters documentation .
Field formatter:

A field formatter uses a field name of the object to display the value.

[source]="personList1"; formatBy="name" Value: {{ value9 | json }}
Locale formatter:

A locale formatter, formats the value according to the current locale.
To use a locale formatter, specify the formatter name prefixed with an @ (e.g., @formatterName).

[source]="personList1" formatBy="@personFormatter1" Value: {{ value18 | json}}
[source]="[true, false]"; formatBy="@boolean:OnOff" Value: {{ value4 | json }}
formatBy="@boolean:EnableDisable" Value: {{ value5 | json }}
Custom formatter:
Using a formatBy function: [formatBy]="functionName"
[source]="personList1" [formatBy]="personFormatterFunc" Value: {{ value17 | json}}

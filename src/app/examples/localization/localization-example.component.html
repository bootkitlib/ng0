Localizing Enums:
ng0TranslateEnum pipe

Example 1 (Done): {{'Done'| ng0TranslateEnum: 'Enum1'}}
Example 2 (Failed): {{'Failed'| ng0TranslateEnum: 'Enum1'}}
Example 3 (Missing enum item): {{'AMissingEnumItem'| ng0TranslateEnum: 'Enum1'}}
Example 4 (empty value localization): {{''| ng0TranslateEnum: 'Enum1'}}
Example 5 (null value localization): {{null| ng0TranslateEnum: 'Enum1'}}
Example 6 (undefined value localization): {{undefined| ng0TranslateEnum: 'Enum1'}}

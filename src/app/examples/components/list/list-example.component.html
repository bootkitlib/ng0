Accessibility

Ensure that the list is accessible to all users, including those using assistive technologies.

Focus:
By setting the focus property, you can control how the list is navigated using the keyboard. focus property can be set to "none", "roving" or "activeDescendant".
  • focus="none": The list will not manage focus for its items.
  • focus="activeDescendant": The list will manage focus for its items using the aria-activedescendant attribute. This is default.
  • focus="roving": The list will manage focus for its items using a roving tabindex.

Format Items:

Format source items by a specific field, a locale formatter or by a custom function.

You have 3 ways to format items:
  • Using object fields: formatBy="fieldName "
  • Using a locale formatter: formatBy="@formatterName "
  • Using a formatBy function: [formatBy]="functionName"
[source]="objectsSource1"; formatBy="name" Value: {{ value10 | json }}
[source]="objectsSource1"; formatBy="@personFormatter1" Value: {{ value29 | json}}
[source]="objectsSource1"; [formatBy]="formatterFunction" Value: {{ value28 | json}}

Bind to LocalDataSource

A LocalDataSource is a data source that can be used to manage the data of the list.
It is a simple wrapper around an array that provides methods to manipulate the data. You can add, remove or update items in the data source and the list will be updated automatically.

Custom item template

Use an ng-template to define a custom item template. The let-option variable is an object with the following properties:

  • index: the item index
  • value: the item value
  • selected: true if the item is selected
  • active: true if the item is active
  • (soon) disabled: true if the item is disabled

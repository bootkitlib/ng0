Formatters are used to format data for display purposes, such as dates, numbers, and currencies.
You can use built-in formatters or create your own custom formatters.

Number formatter

Number formatter formats numeric values for display purposes.

Value = {{123456789 | json}}
Formatter: '#'
Formatted Value = {{123456 | ng0Format: '#'}}
Value = {{12345.12 | json}}
Formatter: '#'
Formatted Value = {{12345.12 | ng0Format: '#'}}
Use with parameters:

You can provide additional parameters to customize the number formatting.

'#':minIntegerDigits:minFractionDigits:maxFractionDigits:useGrouping
Value = {{0 | json}}
Formatter: '#':5:2
Formatted Value = {{0 | ng0Format: '#':5:2}}
Value = {{-1.5 | json}}
Formatter: '#':2:3
Formatted Value = {{-1.5 | ng0Format: '#':2:3}}
Value = {{-1.123456789 | json}}
Formatter: '#':2:undefined:5
Formatted Value = {{-1.123456789 | ng0Format: '#':2:undefined:5}}
Value = {{12345678.123456789 | json}}
Formatter: '#':2:3:undefined:false
Formatted Value = {{ 12345678.0 | ng0Format: '#':2:3:undefined:false}}

Currency formatter

Currency formatter formats numeric values as currency for display purposes.

Value = {{123456789 | json}}
Formatter: '$':'USD'
Formatted Value = {{123456 | ng0Format: '$':'USD':undefined}}
Value = {{12345.12 | json}}
Formatter: '$':'IRR'
Formatted Value = {{12345.12 | ng0Format: '$':'IRR'}}
Use with parameters:

You can provide additional parameters to customize the currency formatting.

'$':currency:notation:minimumIntegerDigits:minimumFractionDigits:maximumFractionDigits:currencySign
Value = {{1.5 | json}}
Formatter: '$':'USD':'standard'
Formatted Value = {{1.5 | ng0Format: '$':'USD':'standard'}}
Value = {{1.5 | json}}
Formatter: '$':'USD':'compact'
Formatted Value = {{1.5 | ng0Format: '$':'USD':'compact'}}
Value = {{1.19 | json}}
Formatter: '$':'USD':'standard':8:3:6
Formatted Value = {{1.19 | ng0Format: '$':'USD':'standard':8:3:6}}
Value = {{12345678.123456789 | json}}
Formatter: '$':'USD':'standard':6:2:4
Formatted Value = {{ 12345678.123456789 | ng0Format: '$':'USD':'standard':6:2:4}}

Date formatter

Date formatter formats date values for display purposes.
A date formatter accepts dates of type Date, number (timestamp), or string (ISO 8601 date string).

  1. Using predefined styles for date and time:
    @:dateStyle:timeStyle:zone:[name, display]:calendar
    • dateStyle: 'short' | 'medium' | 'long' | 'full',
    • timeStyle: 'short' | 'medium' | 'long' | 'full',
    • zone: [name: string, display: string],
    • calendar: string.
      The calendar to use, such as "chinese", "gregory", "persian",
  2. Using custom Intl.DateTimeFormatOptions to specify detailed formatting options.
    '@':{{"{"}} ...Intl.DateTimeFormatOptions {{"}"}}

The date formatter uses the Intl.DateTimeFormat API under the hood, which provides a wide range of options for customizing the date and time formatting.

Default date formatter

By default, the date formatter uses the "medium" style for both date and time components.

Value (Date object) = {{examples.date.value1}}
Formatter: "@"
Formatted Value = {{examples.date.value1 | ng0Format: "@"}}
Value (Epoch) = {{examples.date.value2 | json}}
Formatter: '@'
{{examples.date.value2 | ng0Format: '@'}}
Value (ISO 8601 string) = {{examples.date.value3 | json}}
Formatter: '@'
{{examples.date.value3 | ng0Format: '@'}}
Date only

To display only the date, you can provide the dateStyle parameter.

Value = {{examples.date.value1 | json}}
Formatter: '@':'short'
Formatted Value = {{examples.date.value1 | ng0Format: '@':'short'}}
Value = {{examples.date.value1 | json}}
Formatter: '@':'medium'
Formatted Value = {{examples.date.value1 | ng0Format: '@':'medium'}}
Value = {{examples.date.value1 | json}}
Formatter: '@':'long'
Formatted Value = {{examples.date.value1 | ng0Format: '@':'long'}}
Time only

To display only the time, set dateStyle parameter to undefined and provide a timeStyle parameter.

Value = {{examples.date.value1 | json}}
Formatter: '@':undefined:'short'
Formatted Value = {{examples.date.value1 | ng0Format: '@':undefined:'short'}}
Value = {{examples.date.value1 | json}}
Formatter: '@':undefined:'medium'
Formatted Value = {{examples.date.value1 | ng0Format: '@':undefined:'medium'}}
Value = {{examples.date.value1 | json}}
Formatter: '@':undefined:'long'
Formatted Value = {{examples.date.value1 | ng0Format: '@':undefined:'long'}}
Date and time

To display both date and time, you can provide additional parameters to customize the date formatting.

Value = {{examples.date.value1 | json}}
Formatter: '@':'short':'short'
Formatted Value = {{examples.date.value1 | ng0Format: '@':'short':'short'}}
Value = {{examples.date.value1 | json}}
Formatter: '@':'short':'medium'
Formatted Value = {{examples.date.value1 | ng0Format: '@':'short':'medium'}}
Value = {{examples.date.value1 | json}}
Formatter: '@':'short':'long'
Formatted Value = {{examples.date.value1 | ng0Format: '@':'short':'long'}}
Value = {{examples.date.value1 | json}}
Formatter: '@':'long':'long'
Formatted Value = {{examples.date.value1 | ng0Format: '@':'long':'long'}}
Time zone

You can specify the time zone to format the date and time accordingly.

Value = {{examples.date.value1 | json}}
Formatter: '@':undefined:undefined: ['Asia/Tehran', 'long']
{{examples.date.value1 | ng0Format: '@':undefined:undefined:['Asia/Tehran', 'long']}}
Value = {{examples.date.value1 | json}}
Formatter: ['@', undefined, undefined, ['America/New_York', 'long' ]]
{{examples.date.value1 | ng0Format: ['@', undefined, undefined, ['America/New_York', 'long' ]]}}
Calendar

You can specify the calendar system to format the date accordingly.

Value = {{examples.date.value1 | json}}
Formatter: '@':'long':'medium':undefined:'gregory'
{{examples.date.value1 | ng0Format: '@':'long':'medium':undefined:'gregory'}}
Value = {{examples.date.value1 | json}}
Formatter: '@':'long':'medium':undefined:'persian'
{{examples.date.value1 | ng0Format: '@':'long':'medium':undefined:'persian'}}
Value = {{examples.date.value1 | json}}
Formatter: '@':'long':'medium':undefined:'persian'
{{examples.date.value1 | ng0Format: '@':'long':'medium':undefined:'islamic'}}

Field formatter

A field formatter formats an object by extracting a specific field from it.
This example shows how to use a formatter to display the 'name' field of an object.

Value = {{object1 | json}}
Formatter = 'name'
Formatted Value = {{object1 | ng0Format:'name' | json}}

Array index formatter

An array index formatter formats an array by extracting the element at a specific index.
This example shows how to use a formatter to display the element at index 1 of an array.

Value = {{examples.indexFormatter.value | json}}
Formatter: 2
Formatted Value = {{examples.indexFormatter.value | ng0Format: 2}}

Locale Formatters

A locale formatter formats a value into a human-readable string based on the app Locale.
A Locale formatter starts with the * character followed by the formatter name.
This example shows how to use locale formatters to format enums. .

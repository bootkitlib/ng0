Formatters are used to format data for display purposes, such as dates, numbers, and currencies.
You can use built-in formatters or create your own custom formatters.

Date formatter

Date formatter formats date values for display purposes.
A date formatter accepts dates of type Date, number (timestamp), or string (ISO 8601 date string).

  1. Using predefined styles for date and time:
    ['@', dateStyle?: string , timeStyle?: string, zone?: [name, display?], calendar?: string]
    • dateStyle: 'short' | 'medium' | 'long' | 'full',
    • timeStyle: 'short' | 'medium' | 'long' | 'full',
    • zone: [name: string, display: string],
    • calendar: string.
      The calendar to use, such as "chinese", "gregory", "persian",
  2. Using custom Intl.DateTimeFormatOptions to specify detailed formatting options.
    ['@', {{"{"}} ...Intl.DateTimeFormatOptions {{"}"}}]

To display only the date, you can provide the dateStyle parameter.

Number formatter

Number formatter formats numeric values for display purposes.

Value = {{123456789 | json}}
Formatter: '#'
Formatted Value = {{123456 | ng0Format: '#'}}
Value = {{12345.12 | json}}
Formatter: '#'
Formatted Value = {{12345.12 | ng0Format: '#'}}
Use with parameters:

You can provide additional parameters to customize the number formatting.
Create an array with the formatter name as the first element, followed by the parameters.

['#', minIntegerDigits, minFractionDigits, maxFractionDigits, useGrouping]
Value = {{0 | json}}
Formatter: ['#', 5, 2]
Formatted Value = {{0 | ng0Format: ['#', 5, 2]}}
Value = {{-1.5 | json}}
Formatter: ['#', 2, 3]
Formatted Value = {{-1.5 | ng0Format: ['#', 3, 3]}}
Value = {{-1.123456789 | json}}
Formatter: ['#', 2, undefined, 5]
Formatted Value = {{-1.123456789 | ng0Format: ['#', 2, undefined, 5]}}
Value = {{12345678.123456789 | json}}
Formatter: ['#', 2, 3, undefined, false]
Formatted Value = {{ 12345678.0 | ng0Format: ['#', 2, 3, undefined, false] }}

Locale Formatters

A locale formatter formats a value into a human-readable string based on the app Locale.

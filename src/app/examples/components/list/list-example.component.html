Format Items:

Format source items by a specific field, a locale formatter or by a custom function.

You have 3 ways to format items:
  • Using object fields: formatBy="fieldName "
  • Using a locale formatter: formatBy="@formatterName "
  • Using a formatBy function: [formatBy]="functionName"

Bind to LocalDataSource

A LocalDataSource is a data source that can be used to manage the data of the list.
It is a simple wrapper around an array that provides methods to manipulate the data. You can add, remove or update items in the data source and the list will be updated automatically.
